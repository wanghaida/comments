# Comments

## Usage

Install:

```bash
npm install @uquuu/comments
# OR
yarn add @uquuu/comments
```

Import:

```js
import Comments from '@uquuu/comments';

new Comments({
    appId: '',
    appKey: '',
});
```

Import in Vue:

```js
import Comments from '@uquuu/comments';

router.afterEach(() => {
    new Comments({
        appId: '',
        appKey: '',
    });
});
```

Documents:
